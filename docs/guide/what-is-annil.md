# 什么是 Annil

Annil（安奈儿）是面向微信小程序原生开发的插件，目标是提高原生开发体验,让原生开发能应对复杂项目。

## 核心价值

- **极致的类型安全**

  Annil 提供的 API 使用 TypeScript 编写。 天生拥有更完善的类型提示和类型检查。并持续对 `miniprogram-api-typings` 未覆盖类型也进行了补全或重写。

- **声明式配置**

  相较第三方框架,Annil 沿用了原生小程序组件的声明式配置方式,不增加学习成本。

- **提高组件能力**

  相较原生开发,Annil 提供了更强大的组件能力,如:计算属性(computed)、外部状态绑定(store)、字段监控(watch)等。

- **新的组件构建方式**

  让组件逻辑更清晰,更易于维护。

- **渐进式**

  Annil 提供的 API 基于原生小程序,可以按需引入,无破坏性。

## 适用场景

- 微信小程序原生开发项目。
